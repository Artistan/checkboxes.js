Examples

  • Checking all checkboxes
  • Unchecking all checkboxes
  • Toggling all checkboxes's state
  • Allowing selection ranges
  • Setting max checkable checkboxes

Example #1 Checking all checkboxes

Code:

jQuery(function($) {
    
    $('#btncheck').on('click', function(e) {
        $('#table1').checkboxes('check');
        e.preventDefault();
    });

});
